<h1>Stack:</h1>
Frontend: Vanilla Javascript, HTML, CSS
Backend: Node.js + Express
Database: Neon
External API: ZenQuotes
Hosting: Render

<h1>Description</h1>
This app will help you plan and budget correctly in order not to overspend and live within your means. It is a great tool to know exactly where your money is going and how to meet certain financial goals. It is designed for anyone and everyone.

<h1>Dependencies</h1>
Windows 10 or above if on windows. Mac any text editor of your choice. In this case i am using visual code.

<h1>Brief reflection</h1>
Design Choices:
I chose vanilla Javascript for the frontend just to keep it simple and im more comfortable using it. I used Node.js and Express since its easy to structure routes. There are two main tables -- transactions and goals-- for the database schema. Keeps the data model simple and easy to track. 
Challenges:
When did I not run into challenges. Designing the front end. Hosting it on render. Getting the pie chart to display properly. Tons of hurdles during this project. I solved them by looking at documentation and guides, also just by being calm and looking at it from a fresh perspective. 
Learning Outcomes:
This project gave me hands on experience building a full stack app from scratch. I learned how express routing works, how authentication comes into play/is used, deploying it for the first time. I learned a lot.
Future work:
I was thinking of real time notifications. Real life updates and suggestions pop up on your screen. Also more front end work. That was a struggle dealing with creativity.
